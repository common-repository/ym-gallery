== Description ==

It is a plug-in to add a gallery post. 
I can choose from slide, fancy box, of the type of thumbnail gallery. 
The gallery can be installed in per post.
It is a gallery that is easy to use and simple.


== Installation ==
１．Upload `ym_gallery` to the `/wp-content/plugins/` directory
2.Enable the plug-in
3.Go to the gallery page of YM in the configuration menu 
I want to set the default
4.To publish the article and upload the image to go to the post page
